Create amazing image galleries inside the block editor

== Description ==

Portfolio Blocks is a plugin developed from the ground up for the WordPress Block Editor and Full Site Editor that lets you create
amazing image galleries and portfolios

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
